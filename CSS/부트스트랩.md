# 부트스트랩이란? 

가장 많이쓰는 오픈 소스 프론트엔드 프레임워크. 
스타일이 완성된 여러 컴포넌트들이나 레이아웃 배치를 위한 그리드 시스템 등으로 모바일 우선 반응형 웹 사이트를 만드는데 매우 유용하다.

부트스트랩을 다운로드 받아 사용할 수도 있고 CDN으로 다운로드 없이 사용할 수도 있다.

```
반응형이며 모바일 우선인 웹 프로젝트 개발을 위한 가장 인기있는 HTML, CSS, JS 프레임워크입니다.   - http://bootstrapk.com/
```

# 부트스트랩 주요 기능들 

* 컴포넌트 모음(재사용 가능한 모듈 모음) - 버튼, 네비게이션 바 등 화면 구성을 위해 완성된 요소들을 제공한다.

* 그리드 시스템 - 웹 사이트 레이아웃 설계를 위한 컨테이너, 행, 열을 사용해 화면을 구성하는 방식. flexbox 기반으로 만들어졌으며 반응형이다.

* 기타 등등

부트스트랩 기능 없이도 충분히 웹 페이지를 구성 할 수 있지만 이미 (잘)만들어진 것을 가져다 쓰면 훨씬 빠를 것!
