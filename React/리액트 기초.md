# React란?

리액트는 사용자 인터페이스를 구축하기 위한 자바스크립트 라이브러리이다. 

HTML, CSS, Javascript로도 구축이 충분하나 복잡한 페이지를 좀더 간편하고 가독성있게 구축할 수 있기 때문에 리액트같은 라이브러리를 사용한다.

리액트와 같은 라이브러리를 사용하여 SPA(Single Page Application)을 구축하면 서버로부터 새 페이지를 불러오지 않고 클라이언트 사이드에서 동적으로 페이지가 생성되기 때문에 사용자는 연속되는 페이지 이동에도 모바일 애플리케이션처럼 빠르고 부드러운 사용자 경험을 제공받을 수 있다.

# 컴포넌트(component)란?

리액트는 컴포넌트 기반으로 동작하는데 컴포넌트란 사용자 인터페이스에서 재사용 가능한 빌딩 블록으로, 특정 기능을 수행하는 UI의 부품들이라 생각하면 된다. 예를들면 네비게이션 바, 게시물 블록, 버튼과 같은 것들 모두 하나의 컴포넌트이며 필요에 따라 더 세부적으로 분해할 수도 있다.

리액트가 컴포넌트의 개념을 도입한 것은 재사용성과 코드를 작은 단위로 분할하여 관리할 수 있기 때문. 리액트를 통해 HTML, CSS, Javascript을 결합하여 재사용 가능한 컴포넌트를 만들 수 있으며 
이러한 컴포넌트를 만들기 위해 **선언적 접근 방식(선언형 프로그래밍)**을 사용한다.

즉, 바닐라 자바스크립트로 DOM을 조작하는 것처럼 세부적인 절차를 작성하는 것이 아닌 출력될 사용자 인터페이스 최종 상태(state)를 정의하고, 또 상황에 따라 필요한 상태가 무엇인지 정의하는 것이 중요하다.
